## Summary

Build a premium, modern static landing page for Dr. Shenouda Dental Clinic using React 18, Vite, and Tailwind CSS. The page supports Arabic (RTL) and English (LTR), light/dark themes, advanced SEO with JSON-LD structured data, and rich interactive components (before/after slider, booking form, FAQ accordion, animated stats). All content is sourced from the existing `photos/` directory. No backend, API, or database is used in Phase 1.

## Technical Context

**Language/Version**: JavaScript (ES2022) + React 18.2, JSX via Vite 5

**Primary Dependencies**: React 18, Tailwind CSS 3.4, Framer Motion 11, Google Fonts (Cormorant Garamond, DM Sans, Cairo), Material Symbols Outlined

**Storage**: N/A вЂ” static site with no persistence layer. User preferences (theme, language) stored in `localStorage`.

**Testing**: Browser-based manual validation against acceptance scenarios in spec.md. Optional: Vitest for utility logic (not mandated in Phase 1).

**Target Platform**: Modern evergreen browsers (Chrome, Firefox, Safari, Edge вЂ” latest 2 versions). Mobile-first responsive.

**Project Type**: web-application (single-page static landing page)

**Performance Goals**: First interactive within 3 seconds on simulated 4G; language toggle completes within 500 ms; before/after slider drag latency <100 ms

**Constraints**: No backend/API/database in Phase 1; no IE11 support; bundle size prioritized; images must lazy-load below the fold; no ad-hoc colors вЂ” only design tokens

**Scale/Scope**: Single landing page, 12 sections, bilingual (ar + en), 7 services, 3 before/after cases, 3 reviews, 6 FAQs, 6 booking days Г— 8 time slots
